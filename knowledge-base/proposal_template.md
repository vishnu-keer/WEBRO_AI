# WEBRO — Proposal Template

> The structure the Proposal Generator follows. Edit the skeleton; agents fill it per lead.

1. **Cover** — client name, project title, date.
2. **Understanding** — the client's situation + goals (personalized).
3. **Findings** — key issues from the website audit.
4. **Recommended solution** — what WEBRO will build + why.
5. **Scope & deliverables** — bulleted, concrete.
6. **Timeline** — phases + milestones.
7. **Investment** — package + add-ons (from pricing.md).
8. **Proof** — 1–2 relevant portfolio pieces.
9. **Next step** — single clear CTA.
